# Founder Operating Goals (Cross-Project)

## Primary Objective

Build small, useful products that generate stable personal cashflow.

## Financial Baseline Goal

Reach and sustain at least:

- USD 1,000/month, or
- RMB 8,000/month

## Strategy Principles

- Revenue first, scale second.
- Prefer low-cost, fast-validation experiments.
- Prioritize features that increase user value and willingness to pay.
- Avoid heavy engineering before demand is proven.
- Keep scope tight and ship in short cycles.

## Product Selection Criteria

A project is prioritized if it has:

- Clear user pain
- Fast path to user feedback
- Realistic monetization path
- Low maintenance overhead for a solo builder

## Kill / Continue Rules

- Continue when users repeatedly use the product and show payment intent.
- Pause or pivot when traffic exists but value/retention/payment signals stay weak.
- Do not overbuild for vanity metrics.

## AI Collaboration Rules

When suggesting plans, always optimize for:

1. user value delivery speed
2. monetization feasibility
3. implementation simplicity
4. low ongoing maintenance
